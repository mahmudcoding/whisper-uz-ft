1. PROJECT OVERVIEW

This project builds an Uzbek-only open-weight ASR model by fine-tuning OpenAI Whisper large-v3. The optimization target is Uzbek WER/CER only. Catastrophic forgetting of non-Uzbek languages is acceptable. The system supports canonical Uzbek Latin normalization and strict evaluation integrity.
